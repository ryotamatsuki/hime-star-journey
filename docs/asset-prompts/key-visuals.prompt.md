# 主要キービジュアル生成プロンプト

## 共通指示

手描き絵本風、細いインク線、淡い水彩、ガッシュ調、柔らかい陰影、2.5D見下ろし風の奥行き。主人公ひめはオレンジがかった赤いボブヘア、黒いピナフォア、緑の長袖、しま模様の靴下、紫の靴。相棒シロは小さな白い守り鳥。子ども向けで、温かく少し不思議な雰囲気。

画像内に固定テキストを入れない。UIや会話文はDOMで重ねるため、文字の入る余白を残す。

## 1. タイトル画面

タイトル「ひめの小さな星めぐり」。夕方の道後温泉と松山城のシルエット。手前にひめとシロ、背景に湯けむり、提灯、みかん、星のモチーフ。画面下部にボタンを置ける余白を残す。

## 2. 道後温泉探索画面

夜の道後温泉街。石畳、提灯、木造建築、湯けむり、小さな敵シンボルが見える。ひめとシロが探索している。左上や右下にUIを置ける余白を残す。

## 3. 松山城探索画面

松山城の石段、石垣、城山の森、桜、少し黒い影。怖すぎず、道後温泉より緊張感がある。ひめとシロが奥へ進む構図。

## 4. 通常バトル画面

道後温泉の広場を背景に、ひめとシロ、敵2体が向かい合う。下部にカードUI、上部にHP表示を置く余白。ターゲット選択のハイライトが分かる構図。

## 5. ボスバトル画面

松山城の中庭または天守奥。右側に黒よろいの大将カゲマサ1体、左側にひめとシロ。星封じゲージとカードUIを置く余白。緊張感はあるがホラーにしない。

## 6. 星地図

愛媛県をモチーフにした絵本風の星地図。道後温泉と松山城の星が光り、他の星は暗い。ひめとシロが地図の前にいる。目的地選択UIを重ねられる。

## 7. 会話イベント

道後温泉の夜、ひめとシロが会話する。下部に会話ウィンドウを置く大きな余白。話者名と本文は画像に入れない。

## 8. 旅の手帳

開いた本のUI。左に項目一覧、右にイラストやメモ領域。スタンプ、星、みかん、白鷺、松山城の小物。テキストはDOMで重ねるため、画像内には文字を入れない。
