# boss_kagemasa_sheet 生成プロンプト

## 目的

松山城編のボス「カゲマサ」の立ち絵またはスプライトシートを生成する。

## 共通指示

透明背景PNG。手描き絵本風、細いインク線、淡い水彩。子ども向けで怖すぎない黒よろいの大将。威圧感はあるが、倒すより再封印する相手として表現する。

## デザイン

- 黒い甲冑。
- 大きいが丸みもあり、過度に怖くしない。
- 目は淡く光る。
- 紫色の影をまとっている。
- 古い武将の霊のようだが、最後はしずめる対象。

## 必要アニメーション

- `idle`: 重く構える。
- `attack`: 影を前に伸ばす。
- `charge`: 紫の影を集める。
- `special_attack`: 大きく影を広げる。
- `seal_break`: 星封じで鎧の影がはがれる。
- `vanish`: 光に包まれて再封印される。

## 配置

背景透過。ボス単体でも、スプライトシートでも使えるよう、各ポーズの余白を統一する。
