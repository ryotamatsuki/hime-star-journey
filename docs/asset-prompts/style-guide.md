# アセット共通スタイルガイド

## 画風

- 手描き絵本風。
- 細いインク線。
- 淡い水彩、ガッシュ調の塗り。
- 柔らかい陰影。
- 2.5D見下ろし風の構図。
- 背景は少し高密度、キャラクターは読みやすく単純化。
- 子ども向けだが、幼すぎない。
- 怖い敵も、倒す対象ではなく「しずめる」対象として表現する。

## 色とモチーフ

- UIは美濃紙、木枠、金、星、みかん、白鷺をモチーフにする。
- 道後温泉は湯けむり、石畳、提灯、木造の温泉街、白鷺、みかん色を使う。
- 松山城は石段、石垣、城山の森、桜、井戸、黒い影を使う。
- 松山城は道後温泉より少し冷たく緊張感のある色にする。

## 主人公ひめ

- 小学3年生くらいの女の子。
- オレンジがかった赤いボブヘア。
- 黒いピナフォア。
- 緑の長袖。
- しま模様の靴下。
- 紫色の靴。
- 表情は素直で、少し好奇心がある。

## 相棒シロ

- 小さく丸い白い鳥。
- ひめにしか見えない守り鳥。
- かわいく、少し神秘的。
- 翼は小さく、ふわふわ浮く。

## 禁止・避ける表現

- 写真風、実写風。
- 過度なホラー。
- 過度にリアルな人間。
- 子ども向けに不適切な暴力表現。
- 文字が読めないほど細かいUI。
- 画像内に固定文字を入れること。テキストはDOMで表示する。
- 背景とキャラクターのコントラスト不足。

## 出力の基本

- ゲーム内スプライトはPNG。
- スプライトシートは背景透過。
- UIフレームはテキストなし。
- 背景は16:9を基本にする。
- 実装でCanvas代替図形を描けるよう、アセットIDと用途を明確にする。
