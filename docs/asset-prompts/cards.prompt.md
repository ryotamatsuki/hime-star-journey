# カードアイコン生成プロンプト

## 共通指示

各カードは正方形PNG。手描き絵本風、薄い紙のカード、金の細枠、淡い水彩。バトル画面下部に並べても読みやすい大きなシンボルにする。画像内にカード名の文字を入れない。

## カード一覧

1. `card_mikan_attack.png`
   - 表示名: みかん星アタック
   - みかん色の小さな星が弾ける攻撃カード。

2. `card_shirasagi_ofuda.png`
   - 表示名: 白鷺のおふだ
   - 白鷺の羽とおふだ。浄化、封印補助の印象。

3. `card_dogo_drop.png`
   - 表示名: 道後の湯しずく
   - 湯のしずく、湯けむり、温かい回復の印象。

4. `card_yukemuri_veil.png`
   - 表示名: 湯けむりヴェール
   - 柔らかな湯けむりの膜。防御カード。

5. `card_castle_guard.png`
   - 表示名: 城山のまもり
   - 石垣と城山、強めの防御。

6. `card_star_seal.png`
   - 表示名: 星封じ
   - 星と封印の紋章。ボス専用カード。

## 禁止事項

- 過度にリアルな武器表現。
- 暗すぎてカードの役割が分からない絵。
- 文字入りカード。
