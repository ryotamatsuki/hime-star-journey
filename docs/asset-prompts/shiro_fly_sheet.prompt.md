# shiro_fly_sheet 生成プロンプト

## 目的

相棒シロの探索・会話・バトル共通の浮遊スプライトシートを生成する。

## 共通指示

透明背景PNG。手描き絵本風、細いインク線、淡い水彩。小さく丸い白い守り鳥。ひめにしか見えない少し神秘的な存在。かわいく、怖くしない。

## 必要アニメーション

- `idle`: ふわふわ浮く。
- `follow`: 少し前傾して飛ぶ。
- `surprised`: 羽を広げて驚く。
- `guide`: 小さな星の光を指すように飛ぶ。
- `support`: バトル中にひめを励ます。

## 配置

行ごとにアニメーションを分ける。各フレームは同じサイズ、同じ中心位置。背景透過。
