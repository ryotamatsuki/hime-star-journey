# 通常敵生成プロンプト

## 共通指示

透明背景PNG。手描き絵本風、細いインク線、淡い水彩。子ども向けで怖すぎない。敵は「倒す」より「しずめる」対象として、少し困った表情や不思議な動きを持たせる。バトル画面で2体並べても読めるシルエットにする。

## 敵一覧

1. `dogo_oni.png`
   - 湯どろぼう鬼。小さな鬼、桶を持っている。道後の湯を盗むいたずら者。

2. `dogo_lantern.png`
   - あお提灯。青白く光る提灯のおばけ。ふわふわ浮く。怖すぎない表情。

3. `dogo_armor.png`
   - さびよろい。小さな古い鎧。錆びていて、影に操られている。

4. `dogo_mouse.png`
   - ゆげネズミ。湯けむりをまとった小さなネズミ。すばしっこい。2体並べても読みやすい。

5. `castle_soldier.png`
   - 影足軽。黒い影の小さな足軽。松山城の道をふさぐ。かわいさを少し残す。

6. `castle_oni.png`
   - 石垣鬼。石を背負った鬼。岩を転がす。重そうだがコミカル。

7. `castle_well.png`
   - くらやみ井戸。井戸から出る黒いもや。暗闇状態を使う。不気味すぎない。

8. `castle_crow.png`
   - 黒羽ガラス。黒い羽のカラス。星地図のかけらをくわえる。すばやい印象。

## 禁止事項

- 流血、鋭すぎる牙、過度なホラー表現。
- 背景付き画像。
- 画像内テキスト。
