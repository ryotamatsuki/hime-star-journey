# UIアセット生成プロンプト

## 共通指示

手描き絵本風、美濃紙、木枠、金の細い装飾、星、みかん、白鷺モチーフ。透明背景PNG。画像内に文字を入れない。DOMでテキストを重ねるため、中央に読みやすい余白を残す。

## 必要UI

- `dialogue_frame.png`: 会話ウィンドウ。下部横長、話者名領域と本文領域を分けやすい構造。
- `quest_panel_frame.png`: 探索画面右上または左上に置くクエスト表示枠。小さくても読みやすい。
- `card_frame.png`: バトルカード枠。下部カードUIに並べやすい正方形〜縦長比率。
- `notebook_frame.png`: 旅の手帳画面。開いた本、タブ、スタンプ領域。
- `star_icon_locked.png`: 未解放の星。暗め、輪郭は見える。
- `star_icon_unlocked.png`: 解放済みの星。温かい光。
- `star_icon_cleared.png`: クリア済みの星。明るい金色、達成感。

注: P4.5以降、会話ウィンドウは画像生成せずDOM/CSSで実装する。`dialogue_frame.png` は新規生成対象から外し、会話本文は実テキストとして表示する。

## 禁止事項

- UI内に固定テキストを入れない。
- 装飾が多すぎて文字が読みにくくなる構図にしない。
- 背景色と文字予定領域のコントラストを弱くしない。
