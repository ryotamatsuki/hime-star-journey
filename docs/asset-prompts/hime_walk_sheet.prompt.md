# hime_walk_sheet 生成プロンプト

## 目的

ひめの探索画面用歩行スプライトシートを生成する。

## 共通指示

透明背景PNG。手描き絵本風、細いインク線、淡い水彩。キャラクターは小さく表示しても読みやすいシルエット。足元影は別途Canvasで描くため不要。

## キャラクター

- 小学3年生くらいの女の子。
- オレンジがかった赤いボブヘア。
- 黒いピナフォア。
- 緑の長袖。
- しま模様の靴下。
- 紫色の靴。
- かわいく、読みやすいシルエット。

## 必要フレーム

- `walk_down`: 4フレーム
- `walk_up`: 4フレーム
- `walk_left`: 4フレーム
- `walk_right`: 4フレーム
- `idle_down`: 1フレーム
- `idle_up`: 1フレーム
- `idle_left`: 1フレーム
- `idle_right`: 1フレーム

## 配置

行ごとにアニメーションを分ける。各フレームは同じサイズ、同じ中心位置。背景透過。
