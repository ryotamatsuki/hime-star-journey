# dogo_map_base.png

Generate a hand-drawn picture-book style 2.5D exploration map for a child-friendly game.

Reference:
- docs/visual-reference/key-visuals/02_dogo_exploration.png

Requirements:
- Dogo Onsen inspired hot spring town map, no characters, no enemies, no UI, no text.
- Pale watercolor and gouache, thin ink lines, warm lantern light, readable stone-paved walkable paths.
- Include wooden bathhouse buildings, lanterns, steam vents, bridge, flowers, white heron motif, mikan trees, small star motifs.
- Isometric / 2.5D top-down composition suitable for a player walking through world coordinates.
- Larger than a 1280x720 viewport, composed as a wide exploration map.
- Keep enough clear walkable areas so Hime, Shiro, and enemy symbols remain readable when overlaid.
- Avoid photorealism, horror, UI text, logos, or characters.

Output:
- PNG, wide 16:9 map art.
- Runtime destination: public/assets/generated/backgrounds/dogo_map_base.png
