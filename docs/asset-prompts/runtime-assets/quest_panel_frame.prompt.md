# quest_panel_frame.png

Generate a hand-drawn picture-book style UI frame for an exploration quest panel.

Requirements:
- Parchment center with wooden frame, star, mikan, vine, flower, and white heron motifs.
- Transparent-background PNG preferred; if not possible, use bright chroma-key magenta background.
- Large empty readable center area for DOM text overlay.
- Warm watercolor and gouache, thin ink lines.
- No baked-in text, no buttons, no characters.

Output:
- Runtime destination: public/assets/generated/ui/quest_panel_frame.png
