## Prompt

Use case: illustration-story

Asset type: runtime game background for a 1280x720 Canvas battle screen

Primary request: Create a hand-drawn children's picture-book style battle background for the Dogo Onsen normal battle screen in Hime's small star journey.

Scene/backdrop: Dogo Onsen stone-paved courtyard at twilight, wooden hot spring buildings, warm lanterns, soft steam rising, subtle bridge railing and plants at the edges, small star and mikan motifs hidden in decorations.

Style/medium: pale watercolor and gouache, thin ink outlines, warm hand-painted storybook game art, 2.5D depth, kid-friendly, not scary.

Composition/framing: 16:9 landscape. Leave a clear open staging area on the lower-left for Hime, and one or two clear enemy staging areas on the right. Leave the bottom 22% visually calm and low-detail so DOM card UI can sit over it. Leave upper center readable for a battle message panel. Stronger contrast behind character areas, softer details behind UI areas.

Lighting/mood: warm lantern light, gentle steam, evening blue-violet sky, cozy adventure mood.

Constraints: background only. No characters, no enemies, no cards, no UI frames, no HP bars, no text, no readable signs, no labels, no watermarks. Do not copy the reference image directly. Make it an original runtime background designed for overlaying game UI and characters.

Avoid: photorealism, horror, cluttered foreground, large blank flat color blocks, in-image text, battle cards, portraits, existing UI elements.
