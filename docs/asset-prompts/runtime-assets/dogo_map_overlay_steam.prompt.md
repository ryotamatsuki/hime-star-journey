# dogo_map_overlay_steam.png

Generate a transparent steam and light overlay layer for the Dogo Onsen 2.5D exploration map.

Reference:
- docs/visual-reference/key-visuals/02_dogo_exploration.png

Requirements:
- Transparent-background PNG style asset.
- Thin soft hot-spring steam, warm lantern glow, tiny gold star particles, very light mist.
- Same general 16:9 composition as the base map, but usable as an animated overlay in Canvas.
- Keep opacity gentle so characters, enemies, and UI remain readable.
- Hand-drawn picture-book style, pale watercolor and gouache, no text, no characters, no enemies.
- Use bright chroma-key green background only if true transparency is unavailable, so it can be removed after generation.

Output:
- Runtime destination: public/assets/generated/backgrounds/dogo_map_overlay_steam.png
