# dogo_armor.png

Generate a single hand-drawn picture-book style enemy symbol for Dogo Onsen.

Requirements:
- "Sabi yoroi": a small old rusty armor that has started moving.
- Child-friendly, not horror, looks like something to cleanse rather than destroy.
- Pale watercolor and gouache, thin ink lines, readable armor silhouette at small size.
- Transparent PNG preferred; if not possible, use bright chroma-key green background.
- No text, no UI, no scene background.

Output:
- Runtime destination: public/assets/generated/enemies/dogo_armor.png
