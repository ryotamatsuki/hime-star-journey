# dogo_mouse.png

Generate a single hand-drawn picture-book style enemy symbol for Dogo Onsen.

Requirements:
- "Yuge nezumi": a small cute mouse-like enemy wrapped in soft hot-spring steam.
- Designed so it can also represent a two-mouse encounter.
- Pale watercolor and gouache, thin ink lines, readable silhouette at small size.
- Transparent PNG preferred; if not possible, use bright chroma-key green background.
- No text, no UI, no scene background.

Output:
- Runtime destination: public/assets/generated/enemies/dogo_mouse.png
