## Prompt

手描き絵本風、水彩とガッシュ、細いインク線。道後温泉探索画面で使う会話フレームと目的パネル。美濃紙の淡い背景、木枠、金の細い装飾、星、みかん、白鷺、湯けむり、石畳のモチーフを控えめに使う。DOM文字を重ねても読みやすい広い余白を残す。角は丸すぎず、装飾的だが情報パネルとして見やすい。文字なし。写真風、読みにくい細密装飾は避ける。
