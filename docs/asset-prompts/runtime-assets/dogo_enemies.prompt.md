## Prompt

透明背景。手描き絵本風、水彩とガッシュ、細いインク線。道後温泉探索画面に置く敵シンボル用の単体イラスト。湯どろぼう鬼は小さく丸い鬼で、怖すぎず、湯けむりとみかん色の差し色を持つ。あお提灯は提灯と小さな影の妖怪風で、温泉街の灯りに馴染む。ゆげネズミは湯けむりをまとった小さなネズミで、子ども向けのかわいさを残す。2.5D見下ろし探索で視認しやすいシルエット。文字なし。写真風、リアル観光写真風、ホラー過多は避ける。
