# dogo_lantern.png

Generate a single hand-drawn picture-book style enemy symbol for Dogo Onsen.

Requirements:
- "Ao chochin": a blue-white floating lantern spirit.
- Cute and child-friendly, magical rather than scary.
- Pale watercolor and gouache, thin ink lines, readable lantern silhouette at small size.
- Transparent PNG preferred; if not possible, use bright chroma-key magenta background.
- No text, no UI, no scene background.

Output:
- Runtime destination: public/assets/generated/enemies/dogo_lantern.png
