# MVP詳細GDD

## 目的

このGDDは、企画書をもとにMVPで実際に遊べる内容を具体化する文書です。対象範囲は、道後温泉エリア、松山城エリア、星地図、探索、会話、カード式ターン制バトル、旅の手帳、オートセーブです。

## コアループ

1. 星地図または探索画面で目的を確認する。
2. エリア内を歩いて探索する。
3. NPC、調べ物、敵シンボルに接触する。
4. 敵シンボル接触時は `encounterId` から戦闘を開始する。
5. カードを選び、必要に応じてターゲットを選ぶ。
6. 敵をしずめると進行フラグ、星、手帳項目、カード解放が更新される。
7. 星地図に戻り、次の目的地を選ぶ。

## ストーリー正本

P5.9以降の実装では、以下をMVPストーリーの正本として扱います。

1. ひめは家族旅行で道後温泉へ向かう。
2. 出発前に、おばあちゃんからみかん星のペンダントをもらう。
3. ペンダントは白鷺のお守りであり、土地の星を映す星地図の器である。
4. 道後温泉の夜、ひめはペンダントに宿っていた守り鳥シロと出会う。
5. カゲマサの手下がペンダント中央の「みかん星の核」を奪い、外枠と鎖はひめの手元に残る。
6. みかん星の核を奪われたことで、星地図は弱くしか開けず、星のカードも弱い状態になる。
7. 道後の湯がぬるくなり、提灯が青く光り、湯けむりがざわつき、松山城の方角に黒い影がのびる。
8. ひめはペンダントの光を取り返すため、シロを助けるため、道後温泉と松山城を元に戻すために冒険へ出る。
9. 道後温泉をクリアすると、弱い星地図機能が開き、松山城が次の目的地として解放される。
10. 松山城でカゲマサを再封印し、みかん星の核と城の星を取り戻す。
11. MVP終了時、ペンダントの星地図には未解放の空白の星が残る。

カゲマサはMVPの章ボスであり、物語全体の最終黒幕ではありません。全体黒幕候補の「くろぼし」は、MVPでは完全に正体を明かさず、次の冒険へつながる謎として残します。

## エリア設計

### 道後温泉

道後温泉は、ひめがシロと出会い、ペンダント中央のみかん星の核を奪われる導入エリアです。P6では、道後温泉の異変と「ペンダントの光を取り返す」という最初の目的を短い会話と探索目標で示します。

道後温泉導入で必ず伝える内容:

- ひめはおばあちゃんから託されたペンダントを大切にしている。
- ひめの家は、忘れられた星守りの家系につながっている。
- シロはペンダントに宿っていた守り鳥で、ひめにしか見えない。
- みかん星の核を奪われたことでシロが弱り、道後温泉に異変が起きた。
- ペンダントの外枠と弱い星地図機能は残っているため、P6以降も星地図とカードを使える。
- ひめの当面の目的は、ペンダントの光を取り返し、シロを助け、道後温泉を元に戻すこと。

### 道後温泉クエスト

道後温泉クエストでは、湯どろぼう鬼、あお提灯、さびよろい、ゆげネズミをしずめながら、道後の異変を少しずつ戻します。カード解放はストーリー進行と結びつけます。

| 進行 | 内容 | 報酬・変化 |
|---|---|---|
| 導入 | シロと出会い、みかん星の核を奪われる | 目的「ペンダントの光を取り返す」 |
| 初回戦闘 | 湯どろぼう鬼をしずめる | みかん星アタックの理解 |
| 提灯通り | あお提灯をしずめる | 白鷺のおふだの理解 |
| 路地裏 | さびよろいをしずめる | 湯けむりヴェールの理解 |
| 任意戦闘 | ゆげネズミをしずめる | 複数敵の練習 |

松山城は、道後温泉の異変の先に黒い影が伸びている理由を追う後半エリアです。ひめは、みかん星の核を取り戻す手がかりと、道後温泉を完全に戻すための次の目的地として松山城へ向かいます。

### カゲマサ戦とMVP終了演出

カゲマサ戦は、MVPの章ボス戦です。カゲマサは最終黒幕ではなく、くろぼしの影響を受けた存在、またはくろぼしにつながる謎を示す存在として扱います。

- ボス戦はカゲマサ1体。
- 星封じカードで再封印する。
- 再封印後、ひめはみかん星の核を取り戻し、ペンダントの光を回復する。
- 道後の湯けむりと松山城の星が戻る。
- エンディングでは、星地図に未解放の空白の星が残る。
- 「次は、どこの星を探しに行く？」という余韻で終了する。

## バトルルール

- 味方は `partyMembers: BattleActor[]`、敵は `enemies: BattleActor[]` で管理する。
- MVPの `partyMembers` はひめ1人だけ。
- 攻撃カードで敵が複数いる場合はターゲット選択UIを表示する。
- 攻撃カードで敵が1体だけの場合は自動選択してよい。
- 回復・防御カードはひめ自身に即時適用する。
- 敵全員のHPが0以下になったら勝利。
- ひめのHPが0以下になったら敗北。
- ボス戦ではカゲマサ1体を相手にし、星封じゲージを削り切ることで再封印する。

## 旅の手帳

MVPの旅の手帳は任意閲覧画面です。読まなくても進行可能にします。

- 土地の星: 湯の星、城の星
- カード: 解放済みカード一覧
- 地域メモ: 道後温泉、松山城、白鷺、石垣、地図
- あらすじ: 次の目的と直近の出来事

## プレイテスト観察項目

- 最初の目的が「おばあちゃんにもらったペンダントの光を取り返すこと」として分かるか。
- ペンダントが、おばあちゃんから託された大切なものだと分かるか。
- シロがペンダントに宿っていた守り鳥で、助けたい相手だと分かるか。
- 道後温泉を元に戻す目的が分かるか。
- 松山城へ向かう理由が、黒い影とみかん星の核の奪還に結びついているか。
- 移動操作で迷わないか。
- 敵が何をしているか分かるか。
- 複数敵で攻撃対象を選べるか。
- HPとMPを見て判断できるか。
- 1対2戦闘が難しすぎないか。
- 道後温泉と松山城の違いが伝わるか。
- しずめた敵が消えることに達成感があるか。
- カゲマサが怖すぎず、最後の相手として印象に残るか。
- MVP終了後に「次はどこへ行くのか」と感じられるか。
